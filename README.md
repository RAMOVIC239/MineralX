# 🌍 MineralX Website

MineralX is an educational website that explains mineral extraction methods, mining technology, and sustainable mining practices.

---

## 📌 About the Project
This project is built for learning purposes to demonstrate:
- HTML structure
- CSS styling
- JavaScript interactivity
- Multi-page website design

---

## 🧰 Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Google Fonts (Poppins)

---

## 📂 Website Pages

- 🏠 Home (index.html)
- ℹ️ About (about.html)
- ⛏️ Extraction Methods (extraction.html)
- 🖼️ Gallery (gallery.html)
- 📞 Contact (contact.html)

---

## 🚀 Features

✔ Responsive navigation bar  
✔ Hero section with background image  
✔ Animated hover cards  
✔ Mining methods explanation  
✔ Mineral image gallery  
✔ Contact form (JavaScript validation)  
✔ Mobile-friendly layout  
✔ Clean modern UI design
